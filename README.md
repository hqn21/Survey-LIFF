# survey-liff
Google Sheets API 與 LIFF API 串接範例。

## 前置套件

```bash
composer require google/apiclient:^2.0  # Google Client Library
```
## 網頁架構

```
survey-liff/
│
├── js/
│   └── liff.js           ｜ LIFF API 主要檔案
│
├── googleSheetsAPI.php   ｜ Google Sheets API 主要檔案
│
├── index.php             ｜ 主頁檔案
│
└── signin.css            ｜ Bootstrap 提供之範例 CSS
```
